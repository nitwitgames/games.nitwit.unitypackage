# Skeleton Unity Package

A bare bones skeleton to speed up creation of custom Unity packages for re-usable scripts, editor customisations and game templates.

### Demo

You can use the package manager within Unity to do the following:

* Select `Add package from git URL..`
* Enter `https://github.com/nitwitgames/games.nitwit.unitypackage.git`
* Click `Add` and the package will import and display information
